prefixer.js

prefixer.js is a lightweight library for fixing indentation in <pre> elements.

prefixer-dist.js's file size is not quite figured out yet.

prefixer.js's file size is something we aren't sure about.

It's recommended to view this page's source while looking at the demos!


        // wow. such indent
        // i can haz fix?
        prefix();
        // yay. indent is disappear
      

        // prefixer works with in-code indents
        let foo = 'baz';
        console.log(foo); // -> baz
        {
          let foo = 'bar';
          console.log(foo); // -> bar
        }
        console.log(foo); // -> baz
      

        // unless you make it greedy
        prefix({isGreedy: true});
        {
          // this is an indented block
          let foo = 'baz';
        }
      

        // if just one of your <pre>s hate prefixer
        // it's okay, just tell prefixer not to highlight them
        prefix({noQuery: '#demo-no-prefixer'});

        // noQuery can also take an array as its value
      

        <!--
          these examples use prism.js to syntax-highlight
          prefixer.js should work with just about any
          syntax highlighter; just make sure you put the
          prefix calls before the syntax highlighting calls
        -->

        <script>prefix();</script>
        <script src='prism.js'></script>

        <!--
          if you put prefix calls after, hopefully nothing
          will happen

          but some frameworks might do the formatting right
          away

          which would make all the formatting ded D:
        -->
