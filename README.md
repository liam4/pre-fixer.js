# prefixer.js

fix your `<pre>`s, easy

## Usage

    <head>
      <script src='prefixer.js'></script>
    </head>
    <body>
      <pre><code>
        // ew. too many indents w/o prefixer.js
        console.log('2bad4me');
      </code></pre>
      <script>
        // FIX IT!
        prefix()
      </script>
    </body>
